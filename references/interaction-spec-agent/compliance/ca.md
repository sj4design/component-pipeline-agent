# Canada Compliance — AODA + ACA

> Loaded with `--compliance=ca`. Supplements WCAG 2.2 AA baseline.

## AODA (Accessibility for Ontarians with Disabilities Act)

- Applies to: organizations operating in Ontario (public + private with 50+ employees).
- Technical standard: **WCAG 2.0 AA** (IASR, O. Reg. 191/11).
- Enforcement: compliance orders, administrative penalties, fines up to $100,000/day.

### Requirements

| Area | Requirement |
|---|---|
| Web content | All public-facing web content and web apps MUST conform to WCAG 2.0 AA. |
| New content | New content must be accessible from publication. |
| Feedback | Accessibility feedback process MUST be available. |
| Training | Staff must receive accessibility training. |

## ACA (Accessible Canada Act)

- Federal legislation (2019). Applies to federally regulated organizations.
- Scope: banking, telecommunications, transportation, broadcasting, federal government.
- Creates the Canadian Accessibility Standards Development Organization (Accessibility Standards Canada).
- Technical standards still being developed — currently references WCAG 2.1 as guidance.

### Impact on interaction specs

| Area | Requirement |
|---|---|
| Keyboard | All interactive elements keyboard operable. Mirrors WCAG 2.1.1. |
| Forms | Accessible forms with proper labels, error identification. |
| Mobile | Increasingly expected to be accessible (no specific mobile standard yet). |
| Procurement | Federal procurement must consider accessibility. |

## Provincial variations

| Province | Standard | Notes |
|---|---|---|
| Ontario | WCAG 2.0 AA (AODA) | Most specific enforcement |
| Nova Scotia | Accessibility Act (2017) | Framework in development |
| Manitoba | Accessibility Act (2013) | Standards in development |
| British Columbia | Accessibility Act (2021) | Standards in development |
| Federal | ACA + WCAG 2.1 guidance | Cross-cutting |

## Documentation requirements

- **Accessibility plan**: Multi-year plan describing how barriers will be removed.
- **Compliance report**: Filed with Ontario government (AODA) on schedule.
- **Feedback process**: Documented and publicly available.
