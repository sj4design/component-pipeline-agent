# InlineEdit

## Not Present
Twilio Paste does not include a dedicated InlineEdit component. Inline editing in Twilio Console interfaces (editing phone number names, renaming resources) is typically handled via custom EditableText patterns combining Paste's Text, Input, and Button components. A click-to-edit pattern can be composed from existing Paste primitives without a dedicated component.
