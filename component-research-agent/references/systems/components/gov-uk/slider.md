# Slider

## Approach
GOV.UK Design System does not include a slider/range input component. For numeric value selection, GOV.UK uses text inputs with appropriate type (number) or select components for discrete ranges. Sliders have accessibility concerns (they require precise motor control) and are not appropriate for government services that must serve users with a wide range of motor abilities.

## Key Decisions
1. **No slider** (HIGH) — Motor accessibility requirements make sliders problematic for government services. Text inputs or select dropdowns are used instead.
