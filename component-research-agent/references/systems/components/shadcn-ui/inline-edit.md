# InlineEdit

## Not Present
shadcn/ui does not include a dedicated InlineEdit component. Click-to-edit patterns are composed from shadcn/ui Input, Button, and conditional rendering. A common pattern is to toggle between a Typography display element and an Input field on click, using React state to manage edit mode. Community recipes exist but there is no official InlineEdit component.
