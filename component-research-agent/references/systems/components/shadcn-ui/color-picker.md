# ColorPicker

## Not Present
shadcn/ui does not include a dedicated ColorPicker component. Color input needs can be handled via native input[type=color] with shadcn/ui styling, or via third-party libraries like react-colorful integrated with shadcn/ui's Popover component. Community recipes exist for building color pickers using shadcn/ui Popover + Input patterns.
