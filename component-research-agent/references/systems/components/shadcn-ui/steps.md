# Steps

## Not Present
shadcn/ui does not include a dedicated Steps/Stepper component in its core library. Multi-step wizard patterns are built by composing shadcn/ui primitives. The community ecosystem (shadcn-extension, stepperize) provides Stepper components built with shadcn/ui styling conventions. For documentation-style numbered steps, a simple ordered list with Tailwind styling is the conventional approach.
