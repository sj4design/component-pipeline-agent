# Steps

## Not Present
GitHub Primer does not include a dedicated Steps/Stepper component. GitHub's product UX typically avoids multi-step wizard flows in favor of single-page progressive disclosure. Multi-step processes (like repository setup or organization creation) are implemented as separate pages rather than a tracked step indicator component. Step-like navigation can be approximated with Primer's UnderlineNav or custom implementations.
