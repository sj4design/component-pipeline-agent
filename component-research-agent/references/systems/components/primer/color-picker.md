# ColorPicker

## Not Present
GitHub Primer does not include a dedicated ColorPicker component. GitHub's product primarily uses color in label creation (issue/PR labels), where color is selected from a predefined palette or via hex input in a simple custom implementation. This does not require a full ColorPicker component — label color selection is a product-specific pattern handled outside of Primer's core component library.
