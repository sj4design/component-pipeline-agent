# TimePicker

## Not Present
GitHub Primer does not include a dedicated TimePicker component. GitHub's UI rarely requires time input — most scheduling-related interfaces (Actions cron, release scheduling) use text-based cron expressions or date-only inputs. Time input needs are handled via native input[type=time] with Primer's TextInput styling wrapper when required.
