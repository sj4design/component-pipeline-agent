# Form

## Approach
Wise's Form organizes financial data entry flows — transfer details, recipient information, verification forms. Financial form accuracy is critical so validation feedback is emphasized. Clean, focused form patterns aligned with Wise's UX principle of reducing friction.

## Key Decisions
1. **Financial data accuracy** (MEDIUM) — Forms designed for accurate financial data entry with clear validation.
2. **Minimal friction** (MEDIUM) — Wise's UX principle of reducing steps and confusion in financial forms.
3. **Progressive disclosure** (LOW) — Complex financial forms may use progressive disclosure.

- Standard form layout props

- **Screen reader**: Label associations; error messages
