# Menu / Dropdown

## Approach
Wise's Menu is used for account navigation, transfer options, and contextual actions in their financial product. Menus in financial products need clear, unambiguous action labels given the stakes of financial operations. Clean minimal styling consistent with Wise's product.

## Key Decisions
1. **Financial action clarity** (MEDIUM) — Menu items use clear, unambiguous text labels for financial operations.
2. **Account navigation** (MEDIUM) — Navigation menus for multi-account management.
3. **Minimal styling** (LOW) — Consistent with Wise's aesthetic.

## Notable Props
- Standard menu trigger and items

## A11y Highlights
- **Keyboard**: Arrow key navigation; Escape close
- **Screen reader**: Menu role and item announcements
- **ARIA**: Standard menu ARIA

## Strengths & Gaps
- **Best at**: Financial product menu patterns
- **Missing**: Low confidence — limited documentation
