# Empty State

## Approach
Radix UI does not provide an Empty State component. Empty states are highly product-specific (the illustration, copy, and CTA depend entirely on the context) so Radix appropriately leaves this to consumers. Empty states are built by composing Radix primitives (Heading, Text, Button, an illustration from the icon set) with custom layout.
