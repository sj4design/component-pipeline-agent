# TimePicker

## Not Present
REI Cedar does not include a dedicated TimePicker component. REI's retail and outdoor activity contexts do not heavily feature time-specific input. Time input needs are handled via Cedar's CdrInput component with type="time" when required. Verify in Cedar's current documentation for any newer additions.
