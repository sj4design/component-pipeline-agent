# ColorPicker

## Not Present
REI Cedar does not include a dedicated ColorPicker component. REI's retail context uses color selection for product variants (gear colors), but this is handled via product-specific swatch/option selection components rather than a generic color picker. Freeform color input is not a common REI retail use case. Verify in Cedar's current documentation.
